EasyIDE Library API for Version 2.1

程序简介 基础函数 字符串操作 文件操作 数组操作 解析XML文件 数据操作 翻页操作
程序简介当前版本：EasyIDE Library 2.1。

EasyIDE Library 帮你解决了ASP开发中百分之九十以上的问题，在 EasyIDE Library 支持下，你可以用少量的代码，完成传统ASP开发方法中需要写大段的代码才可以完成的功能。

具体地，EasyIDE Library 包含了字符串操作、磁盘文件操作、数组操作、XML解析操作、数据库操作以及其他很多常用的、已被其他语言作为内置函数的操作，甚至实现对ASP表现层和逻辑层分离的支持：include函数为你实现了稳定的动态包含文件方案，而ob_get_contents函数可以返回ASP页面执行后输出的HTML内容，基于这两个函数和其它辅助函数，你可以使用非标签替换的方法来开发自己的ASP模板引擎。此外str_encode和str_decode也为你提供了ASP版本的字符串可逆加密方案。

作为服务器端动态WEB脚本技术的老前辈，ASP在当今的大型WEB应用中的确已经力不从心了。但是谁都无法否认，在Windows平台的中小型项目快速开发方面ASP的方便、灵活和可控性是其他技术都无法比拟的，这也是ASP一直存在到今天的理由之一。简单易学而且实用的ASP脚本让很多人轻松地迈进了WEB编程的大门，也让很多以ASP为阶梯而进入其他编程技术领域的程序员们一直都没有疏远过这个老朋友，EasyIDE Library 的作者，也正是这其中的一员。

EasyIDE Library 的前身是 EasyIDE Framework V2.0，作者原本打算将这套程序完善成一款ASP开发框架，并且计划基于此程序开发一套免费的CMS程序，但是事情在发展中是不断变化的，2009年，作者投奔了 C/C++ 阵营，不再从事WEB应用方面的开发，因此将 EasyIDE Framework 更名为 EasyIDE Library ，以后不再进行功能上的增强，但是Bug修正还是会不断进行的。

执行指定的ASP脚本并返回执行后待输出的HTML内容。待执行脚本中可以使用当前脚本中的所有资源。

ado_query(byval sql)

以使用最频率的方式执行SQL语句，包括查询性和非查询性语句。函数内部默认设置ADO.Connection连接对象为conn(请先建立此连接)，设置游标类型cursortype为3，设置锁定类型locktype值为1，即读取方式为：

如果要自行设置这些参数请使用 ado_iquery 函数。

pageturner_handle(sql,field_id,page_size)

EasyIDE 翻页预处理函数。支持百万级数据的高效翻页处理。函数内部默认设置的ADO.Connection连接对象为conn(请先建立此连接)。如果要自行定义连接，请使用函数 pageturner_ihandle 。

返回值

长度为4的数组。数组元素分别为：Recordset记录集、当前页面page值、总页数、总记录数。

参数

sql:读取数据的SQL语句。
field_id:读取的数据表主表的索引字段名。page_size:页面大小，即每页显示的记录数。

示例代码 读取所有类别ID为9的新闻并分页(每页显示5条记录)输出其标题：dim sql : sql = "select ns_title from news_list where ns_typeid = 9"
dim page_size : page_size = 30
dim pager : pager = pageturner_handle(sql,"ns_id",page_size)
dim rs : set rs = pager(0)
echo "<ul>"
while not rs.eof
  echo "  <li>"rs("ns_title")&"</li>"
  rs.movenext
wend
echo "</ul>"
rs.close : set rs = nothing 运行结果

  <li>新闻标题一</li>
  <li>新闻标题二</li>
  <li>新闻标题三</li>
  <li>新闻标题四</li>
  <li>新闻标题五</li>
</ul>

pageturner_show(page,page_num,page_sum,page_size,page_len)

pageturner_handle 的辅助函数，用于返回EasyIDE的默认翻页条。查看翻页条的显示效果参见www.EasyIDE.cn。你也可以查看该函数的内部代码，根据代码注释来编写自己的翻页条。

返回值

翻页条字符串。

参数

page:URL翻页参数，函数 pageturner_handle 返回数组的第二个元素(索引值为1)。
page_num:总页数，函数 pageturner_handle 返回数组的第三个元素(索引值为2)。
page_sum:总记录数，函数 pageturner_handle 返回数组的第四个元素(索引值为3)。
page_size:第页显示的记录数，与函数 pageturner_handle 的第四个参数使用同一个变量。
page_len:翻页条的半径长度，即翻页条每次显示page_len*2个页码。

.page_turner a:link, .page_turner a:visited{text-decoration:none; font-weight:normal;}
.page_turner a:hover, .page_turner a.c{background:#c1d9f3; color:#fff;}
.page_turner cite{padding:0;} 运行结果
